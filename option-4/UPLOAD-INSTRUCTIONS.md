# Publishing at elizabethjhwon.github.io

Copy `index.html`, `styles.css`, `.nojekyll`, and the complete `papers` folder into the root of your local website repository, then commit and push them.

On GitHub, rename the repository to `elizabethjhwon.github.io`. Under **Settings → Pages**, choose **Deploy from a branch**, select `main` and `/ (root)`, then save.

GitHub will publish the site at <https://elizabethjhwon.github.io>.

The five publication rows link directly to the PDFs in `papers/`. To change a title, description, year, or PDF link later, edit the corresponding `work-row` in `index.html`.
