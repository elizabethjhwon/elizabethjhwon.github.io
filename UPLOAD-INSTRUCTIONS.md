# Publish both design trials

From Terminal, go to your existing GitHub Pages repository:

```bash
cd ~/Downloads/elizabeth-won-website
```

Copy **the contents** of this ZIP's `elizabeth-live-design-trials` folder into that repository. Allow Finder to replace files and merge folders. Then run:

```bash
git add .
git commit -m "Add two live website design trials"
git push origin main
```

After GitHub Pages finishes deploying, open:

- `https://elizabethjhwon.github.io/`
- `https://elizabethjhwon.github.io/option-1/`
- `https://elizabethjhwon.github.io/option-4/`

The first URL is the temporary comparison page.
